# Analysis-of-Rice-straw-Boiler
# 🔥 Rice Straw Boiler Performance Analysis

A machine learning-based combustion efficiency prediction system 
for a rice straw-fired boiler, based on design and simulation research 
published at RDME-2025.

## 🚀 Live Demo
👉 [Open the Dashboard](https://analysis-of-rice-straw-boiler-6hp89keqegbvvzddwryd37.streamlit.app/)

## 📌 Project Overview
- Analyzed boiler performance across 60–100% load conditions
- Built a Random Forest ML model to predict combustion efficiency
- Deployed an interactive dashboard with real-time parameter sliders

## 🔧 Tech Stack
Python · Pandas · Scikit-learn · Streamlit · Matplotlib · Jupyter

## 📊 Key Findings
- Combustion efficiency ranges from 91.1% to 93.2% across load conditions
- Optimal excess air: ~40–50% for maximum efficiency
- Air flow rate is the most influential feature in the ML model
